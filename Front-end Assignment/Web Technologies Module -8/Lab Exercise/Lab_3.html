CSS Margin and Padding Demonstration

Box 1: margin 10px, padding 10px
This box has small margin and padding.
Box 2: margin 30px, padding 30px
This box has larger margin and padding.
Box 3: margin 50px top/bottom, 10px sides; padding 5px top/bottom, 40px sides
This box uses different values for margin and padding on each side.

Notice: Margin pushes elements away from other elements, while padding adds space inside the border of the element.
